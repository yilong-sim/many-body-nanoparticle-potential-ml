## How to extract figures from the paper PDF

The paper is Open Access (CC BY 4.0) — figures are free to reuse with attribution.

**Snipping Tool (simplest, Windows):**
1. Open paper PDF in browser
2. Zoom in on figure until it fills the screen
3. Windows key + Shift + S → drag to select figure
4. Paste into Paint → Save as PNG

**For higher resolution:**
1. Go to [ilovepdf.com/pdf_to_jpg](https://www.ilovepdf.com/pdf_to_jpg)
2. Upload paper PDF → convert at highest quality
3. Download pages → crop each figure using Windows Photos (Edit → Crop)

Target: at least 800px wide so figures render sharply on GitHub.

## How to convert videos to GIFs

1. Go to [ezgif.com/video-to-gif](https://ezgif.com/video-to-gif)
2. Upload your video
3. Settings:
   - Width: 480px
   - FPS: 10–15
   - Trim to 3–5 seconds of clearest assembly behavior
4. Convert → Save
5. Target file size: under 5MB each
