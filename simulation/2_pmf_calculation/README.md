# 2_pmf_calculation

LAMMPS input script for computing two- and three-body potentials of mean
force (PMFs) via the blue moon ensemble method.

## File

- `pmf_calculation.lammps` — LAMMPS run script (annotated)

## Protocol

Three sequential stages:

**Stage 1 — System preparation**
- Energy minimize the replicated system
- Delete the 4th NP (keeps 3 NPs for trimer PMF calculation)
- Write restart checkpoint

**Stage 2 — Equilibration**
- Move NPs to starting separation via linear displacement (`fix move linear`)
- Equilibrate under NVT (Nosé-Hoover thermostat, T = ε/kB)

**Stage 3 — PMF data collection**
- Fix NP cores rigid (`fix rigid`)
- Step NP1 in +x by Δx = 0.0015σ per interval
- Hold rigid for 600,000 steps at each position (1.2×10⁶ total per point,
  force averaged over last 600k)
- Record constraint force and NP separation → `NPforced{N}.dat`

## Reference

Sprik & Ciccotti, J. Chem. Phys. 109, 7737 (1998) — blue moon ensemble

## Output

`NPforced{N}.dat` files — force vs. separation distance at each grid point.
Feed these into `training/extract_pmf.py` to obtain W2(d) and ΔW3.

## Note on atom IDs

Group definitions assume 21,908 atoms per unit cell as generated by
`1_system_setup/generate_lammps_data.py` with the default param.py.
Update group ID ranges if system size changes.
